# Sensible

A Library of simple and smart logics, types, functions, classes, chains(loops), and decorators for data modeling, workflows and development in:
- [Python](./py)
- [Go](./go)

## Core Philosophy
- **Simple**: Minimal surface area, easy to reason about.
- **Smart**: Sensible defaults, ergonomic APIs, and "just works" behavior.
- **Consistent**: Providing a similar experience across different languages while remaining idiomatic.

## Components
- **Types**: Ergonomic and safe types (e.g., Result/Option patterns, smart strings, custom collections).
- **Logics**: Foundational logic for data flows, validation, and error handling.
- **Chains**: Fluent and expressive ways to compose operations and loops.
- **Workflows**: Simple abstractions for defining and running sequential or parallel tasks.
